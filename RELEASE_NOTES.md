# v0.2.2

Artist matching bug fix release.

## Fixed

- When the user explicitly enters an artist, online search results must now match that artist.
- Prevents same-title but wrong-artist matches such as a Tao Zhe live clip being detected as an AI Sun Yanzi version.
- Improved parsing for video titles containing song names inside `《...》`, such as `陶喆《飞机场的十点半》`.
